TIDE：无损 MoE 扩散 LLM 推理加速，无需重训即获 1.5× 吞吐提升
TIDE: Lossless MoE Diffusion LLM Inference — 1.5× Throughput Gain Without Retraining
Research
本周 arXiv 亮点论文 TIDE 提出了一种针对 MoE 扩散 LLM 的 I/O 感知专家卸载系统，利用专家激活在块内的时间稳定性，在不重新训练模型的情况下实现高达 1.5 倍的吞吐量提升。该方法被描述为扩散 LLM 推理的"免费午餐"——适用于 GPU-CPU 混合系统，对当前主流 MoE 架构（如 DeepSeek V4、Qwen 3.7 等）具有广泛适用性。
This week's arXiv highlight TIDE proposes an I/O-aware expert offloading system for MoE diffusion LLMs that exploits temporal stability of expert activation within blocks, achieving up to 1.5× throughput improvement without model retraining. Described as a "free lunch" for dLLM inference, the method is applicable to GPU-CPU systems and broadly relevant to current MoE architectures (DeepSeek V4, Qwen 3.7, etc.).
为何重要：随着 MoE 架构成为旗舰模型标配（DeepSeek V4、Qwen 3.7-Max、GLM-5.1），推理效率是制约部署规模的关键瓶颈。TIDE 的"零成本加速"思路为大规模 MoE 模型的服务化部署提供了直接可用的优化方案，对降低推理成本具有实际意义。
Why it matters: As MoE becomes the default architecture for flagship models (DeepSeek V4, Qwen 3.7-Max, GLM-5.1), inference efficiency is the key bottleneck for deployment at scale. TIDE's "zero-cost acceleration" approach offers a directly applicable optimization for serving large MoE models, with practical implications for reducing inference costs.

本周 arXiv 上的两项重要研究——TIDE（MoE 扩散 LLM 推理加速）和 Mem-π（Agent 自适应记忆）——均提供了开源代码实现。这延续了 2026 年 AI 研究的趋势：高质量论文越来越多地附带可直接使用的开源代码，加速了学术成果向工业实践的转化。对于开发者而言，TIDE 的推理优化可立即应用于现有 MoE 模型部署，Mem-π 的记忆框架可直接集成到 Agent 系统中。
Two significant arXiv papers this week — TIDE (MoE diffusion LLM inference acceleration) and Mem-π (adaptive agent memory) — both ship with open-source implementations. This continues a 2026 AI research trend: high-quality papers increasingly include immediately usable open-source code, accelerating translation from academic results to industrial practice. For developers, TIDE's inference optimizations can be applied to existing MoE model deployments, while Mem-π's memory framework can be integrated directly into agent systems.
为何重要：开源代码的附带发布正在成为 AI 研究的事实标准，这大大缩短了"论文→产品"的周期。对于当前 MoE 架构主导的模型生态和 Agent 系统开发者而言，TIDE 和 Mem-π 提供了两个可以直接落地的优化工具。
Why it matters: Open-source code alongside papers is becoming the de facto standard for AI research, dramatically shortening the "paper → product" cycle. For the MoE-dominated model ecosystem and agent system developers, TIDE and Mem-π offer two directly deployable optimization tools.

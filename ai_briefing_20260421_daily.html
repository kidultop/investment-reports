OpenAI 于 4 月 14 日发布 GPT-6，内部代号"Spud"（土豆）。新模型采用全新 Symphony 全模态架构，5 万亿参数规模，支持 200 万 Token 上下文窗口（约 100 万汉字），原生多模态 Agent 能力。性能较 GPT-5.4 提升约 40%，在代码生成、多步推理和 Agent 任务上显著领先。与此同时，OpenAI 经历了发布前夕的高管动荡和公关危机。
OpenAI released GPT-6 (codename "Spud") on April 14. The new model features the Symphony full-modal architecture, 5 trillion parameters, a 2M-token context window (~1M Chinese characters), and native multimodal agent capabilities. Performance improved ~40% over GPT-5.4 with significant leads in code generation, multi-step reasoning, and agent tasks. Simultaneously, OpenAI experienced executive turmoil and PR crises ahead of the launch.
为何重要：GPT-6 是近两年竞争格局最关键的旗舰发布，200 万 Token 上下文将重塑长文档处理和复杂任务编排的行业标准。
Why it matters: GPT-6 is the most critical flagship release in two years; the 2M-token context window resets industry standards for long-document processing and complex task orchestration.
